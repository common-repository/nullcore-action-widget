Add a widget to your sidebar with a custom action hook.

== Description ==

The NullCore Action Widget plugin adds a widget which can be used to add a custom action hook to your widget areas.

<a href="http://www.nullcore.nl/239/nullcore-action-widget/">More information.</a>


== Installation ==

1. Download and extract the “nullcore-action-widget” folder
1. Upload the “nullcore-action-widget” folder to your WordPress plugin directory, usually “wp-content/plugins”
1. Activate the plugin in your WordPress admin panel
